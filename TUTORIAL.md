### B. Software yang Dibutuhkan
- Windows 7/8/10/11 (64bit recommended)
- Python 3.8+

### C. Install Python
1. Download Python dari python.org

3. CENTANG "Add Python to PATH"
4. Klik "Install Now"
5. Verifikasi: buka CMD ketik `python --version`

### MENU 3 - CHECK BUGS

Bug Hunter Mode - Penjelasan Lengkap:

**A. SQL Injection**
Tanda-tanda website vulnerable:
- URL dengan parameter seperti ?id=1
- Website dengan fitur search
- Login form tanpa proteksi

**B. XSS (Cross Site Scripting)**
Ciri-ciri:
- Input form tanpa filter
- URL parameter langsung ditampilkan
- Comment section

**C. LFI (Local File Inclusion)**
Test payload:
../../../../etc/passwd
../../../../config.php
../../../../.htaccess
**D. Backup File Exposure**
File yang dicari:
backup.zip
backup.sql
config.php.bak
.htaccess.bak

## BAB 5: CARA MENGHINDARI DETEKSI

### A. Sebelum Attack

1. **Clean State**
- Clear browser history
- Clear DNS cache (`ipconfig /flushdns`)
- Restart modem/router

2. **Change Identity**
- Ganti MAC address
- Ganti IP (VPN/Proxy)
- Ganti User-Agent

### B. Selama Attack

1. **Rotate Patterns**
- Jangan attack terus menerus
- Gunakan interval acak

2. **Mix Methods**
- Campur DOS dan DDOS
- Ganti target port

### C. Setelah Attack

1. **Clean Logs**
- Tools otomatis clear logs
- Hapus semua temporary files

2. **Change IP lagi**
- Matikan VPN
- Nyalakan ulang
- Ganti server
